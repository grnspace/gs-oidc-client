Add a “Sign in with GS” button

We provide a “Sign in with GS” button for directing your clients to the GS sign in portal. The button can easily be added to your site using JS or a combination of HTML and CSS.

Without JS

No JS? No problem! Add the button by including the following CSS and HTML in your page.
